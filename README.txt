# CSS101 Exercise 1: Profile Web Page

This repository contains the HTML and CSS source code for a profile web page, mimicking a resume but presented as a web page. The goal of this exercise is to gain hands-on experience with HTML and CSS to create a visually appealing and informative personal profile.

## Project Overview

The profile web page includes various sections to showcase personal information, educational background, work experience, skills, and other relevant details. The CSS styles are designed to enhance the visual presentation and make the page responsive to different screen sizes.
